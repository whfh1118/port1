Project 2: I-Tech Enc 프로젝트 링크 바로가기

  • Overview
    I-Tech는 전문 건설업체로서 회사가 가지고 있는 전문분야를 고객 계열사에 어필할 수 있도록 정보를 리스트로 구성 했습니다.
    Color Identity
    I-Tech의 이름에 걸맞는 기술적인 이미지들에 맞춘 Color를 사용 했습니다.
  • As Is
    단순한 레이아웃 구조와 가독성 떨어지는 컨텐츠 등의 문제점이 있었습니다.
    To Be
    계열사에 어필할수 있을정도의 정보량을 보여주기위한 정보전달 위주의 레이아웃을 구성 했습니다.

  • 정보전달컨텐츠 추가 수정

Web Version - Top Area

  • Main Menu Bar
    기존의 복잡한 컨텐츠 전달방식을 Top영역 NAV BAR에 압축 구성 했습니다.
  • Main Slider
    기존 노후된 Slider Keyvisual 이미지를 최신 버전으로 업데이트하고 Description 기능을 추가 했습니다.
  • Slide Banner
    협력업체들의 로고를 배너에 구성하여 협력 업체 사이트에 접근하기 쉽게 구성 했습니다.

Web Version - Center Area

  • Main Banner
    사업개요에 맞춘 각각의 정보를 리스트화 하여 3종류의 배너로 구성 했습니다.
  • Info Banner
    I-Tech기술의 다양한 정보들을 배치할 배너리스트를 구성 했습니다.
  • Popup
    추가적 아이콘 컨텐츠들을 담을 수 있는 공간으로 팝업창을 미리 구성 했습니다.

Web Version - Bottom Area

  • Vision Banner
    회사 기술의 비전, 및 미션을 담은 배너리스트를 위 Main, Info 배너와 다른 형식으로 구성 했습니다.
  • Affiliate Banner
    상단 제휴리스트 배너외에 하단 컨텐츠에서도 배너를 제공하여 제휴처의 정보를 다시 제공 했습니다.
