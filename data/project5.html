Web Version - Top Area

  • Main Menu Bar
    메뉴 리스트를 누르면 각 컨텐츠로 위치 이동되는 방식으로 구성 했습니다.
  • Main Slider
    ZARA 브랜드에 어울리는 Video로 Keyvisual을 구성했습니다.
  • Responsive Tab
    상단부 반응형 탭부분으로 메뉴리스트 형식을 그대로 탭부분에 구성 했습니다.

Web Version - Center Area

  • Woman Banner
    기존 ZARA홈페이지에서 구성하지 않은 리스트 방식으로 각 카테고리별 리스트타입으로 구성 했습니다.
  • Man Banner
    Woman Banner와 비슷하지만 링크타입으로 정보를 줄수 있는 배너방식으로 구성 했습니다.

Web Version - Center Area2

  • KIDS BANNER
    Kids영역은 Woman Banner와 같은 리스트형식으로 구성했고 리스트가 반응형으로 배치가 재정비되는 구조입니다.
  • ACCESSORY
    다른 배너와 비슷한 반응형 타입으로 구성했습니다.

Web Version - Bottom Area

  • LOGO BANNER
    ZARA 브랜드와 같은 계열 브랜드의 로고로 하단부 배너를 구성했습니다.
  • Contact Us
    ZARA 브랜드에 요청 및 수정사항 등등 고객이 필요한 요청사항을 보낼 수 있는 E-Mail영역을 구성했습니다.
  • Footer Contents
    트위터 , 페이스북등 SNS매체를 통해 접근할 수 있도록 아이콘으로 구성해서 배치했습니다.
