Project 1: Sky Vivaldi 프로젝트 링크 바로가기

  • Overview
    Sky Vivaldi는 펜션업체로서 고객들이 간편하게 접근할 수 있는 정보를 홈페이지에 포함해야합니다. 고객들의 필요정보를 가독성 좋게 리뉴얼 하는게 이번 프로젝트의 목표입니다.
    Color Identity
    여행이라는 느낌에 맞는 색상을 사용하여 산뜻한 느낌을 주었습니다.
  • As Is
    단순한 레이아웃 구조와 가독성 떨어지는 컨텐츠 등의 문제점이 있었습니다.
    To Be
    소비자가 얻고자하는 정보를 보다 쉽게 접근할 수 있도록 배치했고 부족한 컨텐츠를 수정후 추가 했습니다.

  • 효율적 접근 배치컨텐츠 추가 수정

Web Version - Top Area

  • Main Menu Bar
    기존의 복잡한 컨텐츠 전달방식을 Top영역 NAV BAR에 효과적으로 압축하여 소비자들이 원하는 정보를 가독성 좋게 구성 했습니다.
  • Main Slider
    기존 노후된 Slider Keyvisual 이미지를 최신 버전으로 업데이트하고 Pagination 기능과 Description 기능을 추가 했습니다.
  • Notice & Main Banner
    펜션측의 공지사항을 통해 소비자들이 미리 숙지할수있는 정보를 보다 접근하기 쉬운 장소에 배치하여 정보전달력을 높이고 Banner에 숙소 사진을 배치 했습니다.

Web Version - Center Area

  • Main Banner
    호버 기능을 통해 주변관광지의 정보 또한 쉽게 접근 할 수 있게 전달방식을 효율적으로 압축 했습니다.
  • Service Menu
    소비자가 펜션에게 얻고자하는 주요정보를 아이콘으로 제작하여 보다 접근하기 쉽도록 구성 했습 니다.
  • Popup
    다양한 메신저 SNS매체를 통해 또 다른 홍보효과를 기대 할 수 있도록 Popup Bar를 추가 구성 했습니다.

Web Version - Bottom Area

  • Mobile Contents
    핸드폰의 간단한 문자전송으로 펜션의 예약기능과 각종 혜택등을 쉽게 알아 볼 수 있도록 App Contents를 추가 했습니다.
  • Footer Contents
    하단 Footer Contents에서도 상위메뉴에서 지나칠 수 있는 정보와 업체의 간략한 정보, 연락처등을 보기 쉽게 구성 했습니다.

Web Version - Main Area

  • Tab Menu
    네비게이션 영역을 탭 메뉴로압축했고 탭 버튼의 크기를 사용자들의 편의성을 고려해 적절하게 조정 했습니다.
  • Tab Menu - Inner Contents
    심플한 디자인으로 가독성 좋고 깔끔한 느낌을 표현 했습니다.
  • Main Slider
    모바일에 적합한 이미지로 대체했고, 하단부에 가이드를 배치하여 슬라이더 하위 컨텐츠에 접근 할 수 있도록 표현 했습니다.

Web Version - Center Area

  • Main Banner
    다양한 객실 정보와 혜택들을 보기좋게 구성했고, 슬라이더효과와 클릭 시 적용되는 액션을 추가 했습니다.
  • Center Banner
    Vivaldi 팬션을 통한 추가적인 혜택들중 근처 관광지들에 대한 정보를 접근하기 쉽도록 슬라이더 이미지를 통해 구성 했습니다.
  • Quick Banner
    지형약도, 예약, 객실등 고객의 필요한 정보들을 아이콘으로 구성하여 보기좋게 압축 구성 했습니다.

Web Version - Bottom Area

  • Notice
    Notice 영역을 통해서 Vivaldi 팬션측과 고객들의 커뮤니케이션이 가능하도록 구성 했습니다.
  • Footer Area
    Google Map을 통해 팬션의 위치를 보기쉽게 구성했고, footer영역에 꼭 필요한 정보만 압축 구성 했습니다.
