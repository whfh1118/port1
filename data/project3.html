  • 레이아웃 최적화컨텐츠 추가 수정

Web Version - Top Area

  • Main Menu Bar
    메뉴 리스트를 상품 종류로 배치하여 편의성을 높이고 아이콘을 탭을 구성 했습니다.
  • Main Slider
    따뜻한 느낌과 BERBERRY 색감에 맞는 keyvisual을 사용 했습니다.
  • Top Banner
    메인 메뉴 리스트에서 할당되지 못한 컨텐츠를 배너 컨텐츠를 통해 구성 했습니다.

Web Version - Center Area

  • Classic Item Banner
    BERBERRY회사의 Classic 상품을 단독 배너로 구성 했습니다.
  • Video Area
    추가적인 Video기능으로 상품들에 특수 대한 전달력을 높일수 있게 구성했습니다.

Web Version - Bottom Area

  • Weather Contents
    계절별 핫 아이템들을 위한 컨텐츠란과 그 컨텐츠의 정보를 한영역에 구성했습니다
  • Icon Banner
    Icon을 통해 쇼핑정보, 상품 정보, 배송 정보등의 전달력을 강화했습니다.
  • Footer Contents
    상품정보등 중요한 정보들을 제외한 정보를 모아 하단 Footer Contents에 제공 했습니다.

Web Version - Main Area

  • Tab Menu
    네비게이션 영역을 탭 메뉴로압축했고 탭 버튼의 크기를 사용자들의 편의성을 고려해 적절하게 조정 했습니다.
  • Tab Menu - Inner Contents
    심플한 디자인으로 가독성 좋고 깔끔한 느낌을 표현 했습니다.
  • Main Slider
    모바일에 적합한 이미지로 대체했고, 하단부에 가이드를 배치하여 슬라이더 하위 컨텐츠에 접근 할 수 있도록 표현 했습니다.

Web Version - Center Area

  • Main Banner
    다양한 객실 정보와 혜택들을 보기좋게 구성했고, 슬라이더효과와 클릭 시 적용되는 액션을 추가 했습니다.
  • Center Banner
    Vivaldi 팬션을 통한 추가적인 혜택들중 근처 관광지들에 대한 정보를 접근하기 쉽도록 슬라이더 이미지를 통해 구성 했습니다.
  • Quick Banner
    지형약도, 예약, 객실등 고객의 필요한 정보들을 아이콘으로 구성하여 보기좋게 압축 구성 했습니다.

Web Version - Bottom Area

  • Footer Contents
    Pc와 동일 디자인으로 설계했습니다.
  • Login Menu
    Login Menu의 전체 색을 BERBERRY고유 색상으로 설정했고, Login 화면에 필요한 컨텐츠의 크기를 터치가 쉬운 크기로 설정했습니다.
  • Inner Contents
    배너에 상세 페이지를 추가해서 제공하지 못한 정보들을 손쉽게 링크할 수 있도록 수정 했습니다.
