Project 4: Mary Mond 프로젝트 링크 바로가기

  • Overview
    일반 웹 사이트에서 반응형 웹사이트로 컨버팅하며 메인페이지에서 주는 정보전달력을 보다 높이는게 목표입니다.
    Color Identity
    꽃할머니 프로젝트를 하고있는 마리몬드의 스타일을 고려한 색을 사용했습니다.
  • As Is
    데스크탑으로 접근할 수 밖에 없는 구조입니다.
    To Be
    반응형 웹구조로 컨버팅하여 작거나 큰화면에서도 사용자가 볼 수 있도록 했습니다.

Responsive Web - Top Area

  • Main Menu Bar
    Main Menu Bar에서 정보전달을 효과적으로 할 수 있도록 영역을 크게 구성하고 컨텐츠를 추가 했습니다.
  • Main Slider
    Slider 이미지를 계절 상품 이미지로 대체 했습니다.
  • Slider Pagenation
    Main Slider의 Pagenation을 배너 형식으로 구성했습니다. 다른 Pagenation과 동일하게 클릭하면 페이지가 넘어가는 시스템입니다.

Web Version - Center Area

  • Top Banner
    Marymond 상단부 배너에 비교적 중요도가 낮은 Event들을 배치할수 있도록 Main Banner보다 적은 영역을 구성 했습니다.
  • Main Banner
    Marymond Event중 중요도가 높은 컨텐츠를 배치했고, 배경이미지에 맞춘 반응형 배너로 비디오 및 링크 배너를 추가 했습니다.

Web Version - Bottom Area

  • Main Contents
    Marymond에서 기획한 메인 Event를 각각의 배너로 구성했고, 따뜻한 이미지의 색을 사용 했습니다.
  • Footer Contents
    대부분의 footer영역에 들어가는 내용을 보기쉽게 리스트 형식으로 구성했습니다.
